# Evaluation of Mobile Applications for Early Detection of Skin Cancer

This repository contains the experimental materials, dataset references, tested sample images, and evaluation results used in the bachelor thesis titled:

**“Evaluation of Mobile Applications for Early Detection of Skin Cancer”**

The study focuses on comparing publicly available mobile applications designed to support the early detection of skin cancer. The main purpose is to evaluate how consistently these applications respond when the same skin lesion images are tested multiple times under identical conditions.

> **Disclaimer:**  
> This project is for academic and research purposes only.

## Project Overview

Skin cancer, especially melanoma, can be dangerous if it is not detected at an early stage. With the increasing use of smartphones, many mobile applications now allow users to photograph skin lesions and receive a risk-based assessment.

However, these applications may not always provide stable or clear results. The same image can sometimes produce different outputs when tested more than once. This study investigates the practical reliability of such applications from a user-oriented perspective.

The project evaluates selected mobile applications according to the following criteria:

- **Risk-assessment consistency**
- **Processing time**
- **Interface usability**
- **Risk communication clarity**

---

## Aim of the Study

The aim of this study is to evaluate and compare existing mobile applications designed for early skin cancer detection, focusing on their consistency, usability, processing speed, and clarity of information provided to users.

---

## Tested Mobile Applications

The following mobile applications were selected and tested:

| Application | Processing Type | Output Format |
|---|---|---|
| SkinVision | Server-side | Low / Medium / High risk |
| SkiniveMD | Server-side | Multi-condition AI report |
| SkinScreener | On-device | Low / Medium / High risk |

These applications were selected because they are publicly available and provide automated image-based skin lesion assessment.
